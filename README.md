# Deep Learning Notebooks

Public notebook collection covering neural network training, CNNs, MNIST classification, optimization, and model evaluation.

## Notebooks

- `CNNtraining4.ipynb` - CNN model training and evaluation workflow.
- `MNIST-hw4.ipynb` - MNIST classification experiments.
- `3layerRELU.ipynb` - multi-layer neural network experiments with ReLU activations.
- `batch vs stochastic descent.ipynb` - optimization comparison notes and experiments.
- `Handwriting.ipynb` - handwriting classification workflow.

## Tools And Topics

Python, Jupyter Notebook, TensorFlow/Keras, CNNs, MNIST, model training, accuracy evaluation, optimization.
